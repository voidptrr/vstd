# Skills

Each skill is machine-consumable and must live at `.agents/skills/<skill-id>/SKILL.md`.

Required YAML keys in each `SKILL.md` frontmatter:

- `name`
- `description`
- `version`
- `allowed-tools`

Required sections in markdown body:

- `## Overview`
- `## When to Use`
- `## Process`
- `## Rationalizations`
- `## Red Flags`
- `## Verification`
